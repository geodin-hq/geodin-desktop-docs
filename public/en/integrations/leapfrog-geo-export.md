# Export to Leapfrog Geo

<!-- src: adhoc/2026-07-16_website-drift -->

GeoDin includes a built-in Leapfrog exporter, available for the G1 object type (GeoDin
15.2.53 and later). It writes your geological data - drilling data, laboratory results, and
quality-controlled datasets - to a file that Seequent Leapfrog Geo can import directly,
removing manual conversion steps between the GeoDin database and 3D geological modelling in
Leapfrog.

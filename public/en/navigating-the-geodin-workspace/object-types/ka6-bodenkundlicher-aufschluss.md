# (KA6) Bodenkundlicher Aufschluss

<!-- src: adhoc/2026-07-16_website-drift -->

Introduced in GeoDin 15.6 (release 15.6.6.125, June 2026), the KA6 object type supports the
6th edition of the German Soil Survey Guidelines (Bodenkundliche Kartieranleitung, KA6),
mandatory in Germany since July 2024. It covers diagnostic horizons, revised horizon symbols,
the extended substrate system, and the new soil types defined by KA6. Projects following the
5th edition can continue to use the existing KA5 support.

# User Interface

### 1. Databases

* The **Databases** panel is the starting point of your work in GeoDin.
* Each database contains **Projects** (Example: G1\_ASTM\_DEMO\_LasVegas in Figure 1), and each project consists of **Objects** (such as boreholes/ locations and test type), **Measurement Points, Documents**.

#### 1.1 Database Types

GeoDin supports **Client-Server Databases**, which can be hosted on a single computer or across a network.

Databases are color-coded for easy identification:

* **Yellow** – Network-level databases (shared by a group of users for collaboration).
* **Blue** – Local databases (accessible only to the individual user).

{% hint style="info" %}
Note: GeoDin databases are based on **Microsoft Access files**.
{% endhint %}

### 3. Interface Icons

**Bottom Left Icons**

* **Layout Overview** – Provides a quick view of the layouts.
* **Edit Graphics** – Allows you to modify graphical elements within your project.

**Top Right Icons**

* **GeoDin Help** – Opens the help documentation for assistance or simply press **F1.**
* **Information** – Provides information about the version and license validity (Alt + I), SQL Protocol (Alt + S) and access to Log Folder (Alt + H).

**Central Ribbon**

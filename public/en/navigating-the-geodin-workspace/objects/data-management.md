# Data Management

The **Data Management** method is the central interface for entering and editing data for an object (borehole, sample point, monitoring well). It presents the available data tabs based on the object type, and is launched by double-clicking the **Data management** method icon on a selected object.

## Reference: Data tabs

Typical tabs include:

- [General Data](general-data.md) - identifying information, coordinates, project association
- Layer Data - geological layer descriptions, stratigraphy, layer-by-layer entry
- [Sample Data](sample-data.md) - physical samples and their depth intervals
- [Well Design Data](well-design-data.md) - casing, screens, backfill, piezometer details
- Data Sequences - continuous depth-indexed measurements (CPT, geophysical logs)

For the comprehensive object operations workflow - creating, editing, validating, importing, exporting, and reporting object data - see [**Object Operations Reference**](../object-types/object-operations-reference.md).

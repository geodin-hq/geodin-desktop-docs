# Creating Queries

Queries in GeoDin produce filtered views of objects, samples, and measurement data. They appear in the **Object Manager** alongside the default object groupings.

A query is built by defining one or more conditions on object or measurement attributes. The result can be saved as a named query (visible in the Object Manager tree) or as an ad-hoc selection used inside a method.

For the comprehensive reference — full syntax, all options, edge cases — see [**Query Builder Reference**](query-builder-reference.md).

# Data Analysis

Querying, filtering, formulas, and the calculation engine - finding and deriving value from the data you collected.

## Queries

Filter and retrieve objects, measurement points, and layer descriptions by condition.

- [Creating Queries](queries/creating-queries.md) - filter objects and measurement points by conditions.
- [Parameterized Queries](queries/parameterized-queries.md) - queries with placeholders that prompt for values at execution time.
- [SQL and Advanced Options](queries/sql-and-advanced-options.md) - direct SQL entry for logic the visual builder cannot express.
- [Query Builder Reference](queries/query-builder-reference.md) - reference for the Query Builder, including the subquery table list and schema handling.
- [Conditions and Operators](queries/conditions-and-operators.md) - comparison and logical operators used to filter a result set.
- [Layer Queries](queries/layer-queries.md) - query coded layer descriptions such as soil and rock attributes.
- [Complex Layer Queries](queries/complex-layer-queries.md) - advanced layer-query cases building on Layer Queries.

## Formulas

Compute and store derived values from data already in the dataset.

- [Formula Basics](formulas/formula-basics.md) - compute derived values with GeoDin's formula syntax.
- [Formulas in Measurement Values](formulas/formulas-in-measurement-values.md) - apply formulas to measurement-value series.
- [Object Type Formulas](formulas/object-type-formulas.md) - define formulas on a data type's fields.

## Calculation engine

Automated checks, geotechnical analyses, and curve-fitting applied to your data.

- [Data Checks and Validations](calculation-engine/data-checks-and-validations.md) - automated checks and rules applied at data entry.
- [Geotechnical Analyses](calculation-engine/geotechnical-analyses.md) - hydrogeochemical and grain-size analysis methods and their diagrams.
- [Regression and Curve Fitting](calculation-engine/regression-and-curve-fitting.md) - time-series and measurement-value series properties for regression and curve-fitting work.

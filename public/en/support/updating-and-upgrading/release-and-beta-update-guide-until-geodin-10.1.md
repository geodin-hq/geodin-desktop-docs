# Release and Beta Update Guide

#### 1. **Update to the Latest Release Version**

**Online Update**

* Navigate to the **System** tab at the top left of the GeoDin user interface.
* Start the **Online-Update** method under the **System Configuration** node.

**Manual Update for GeoDin 15**

* Contact  [support@geodin.com](mailto:support@geodin.com) for assistance.

**Manual Update for GeoDin 10.1**

1. Download the `geodin.zip` file from:\
   [https://download.geodin.com/geodin/geodin.zip](https://download.geodin.com/geodin/geodin.zip)\
   &#xNAN;_(Older program files are available in the archive:_ [_https://download.geodin.com/geodin/archive/_](https://download.geodin.com/geodin/archive/)_)_
2. Unzip the downloaded archive.
3. Replace the existing `geodin.exe` file in the **BIN** directory of your GeoDin installation with the new one.\
   &#xNAN;_(Tip: Rename or move the existing file beforehand for backup purposes.)_



#### **2. Update to the Latest Beta Version (Until GeoDin 10.1)**

The beta version includes the latest developments and improvements but may contain minor bugs. Please note this before installation.

Steps:

1. Download the `geodin.zip` file from:\
   [https://download.geodin.com/geodin/beta/geodin.zip](https://download.geodin.com/geodin/beta/geodin.zip)
2. Unpack the downloaded ZIP archive.
3. Replace the existing `geodin.exe` file in the **BIN** directory of your GeoDin installation with the new one.\
   &#xNAN;_(Tip: Rename or move the existing file beforehand for backup purposes.)_

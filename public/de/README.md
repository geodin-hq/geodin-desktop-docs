# GeoDin®

GeoDin ist eine Software zur Verwaltung geotechnischer Daten, die sich hauptsächlich auf geotechnische Baugrunduntersuchungen konzentriert. GeoDin unterstützt auch die Verwaltung von Umweltdaten.

Die Hauptmerkmale von GeoDin umfassen:

* Einhaltung von Industriestandards wie ASTM D2487/D2488, BS 5930, EN ISO 14688/14689 und ähnlichen.
* Unterstützung lokaler deutscher Standards wie SEP3 und KA5.
* Anpassbare Datenmodelle und Formeln.
* Visualisierungstools mit kostenlosen Vorlagen.
* Integriertes GIS (GeoDin Maps) für umfassende Analysen und Grafiken.
